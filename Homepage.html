How to Use this Website

Homepage

Click on the words on the top of the page that read, 'The <Code> Dictionary', and that will take you right back to the homepage. If you try it right now, it will work, but you will not notice because you are already on the homepage.

Random + Random All buttons

If you click the button marked 'Random', it will take you to a random tag in the coding language that you are currently on. If you click the button that says 'Random All', it will take you to a random tag within the whole entire website.

Search Bar

The search bar is what you can search different tags or categories in. For example, if I wanted to find the 'print' tag in Python, I would search it up using the search bar, by typing 'print' into the search bar, and pressing enter. Then it will take me to a page that will display all of the results from my query.

HTML, CSS, Javascript and Python buttons

The HTML, CSS, Javascript and Python buttons will lead you to all of the different tags in that coding language that we know. If you click on the HTML button, it will take you to the HTML page, where all of the different tags are listed.

Menu

The menu is the bar that is to the right of the page (it only shows up if you are in a coding language), and it has links to get to different categories in the coding language that you are in.

The Learn button

There is a button that is marked 'Learn', to the right of the button marked 'Python'. That button will take you to lesson that will be in a list of commands that you might not know. Now, if you know the tag, then there will be a button on the bottom that says 'Next >', and if you didn't mean to click next, there will be a button on the other side of the page marked '< Previous'.
